<h1 align="center">🔍 Property Path Inspector</h1>

<p align="center">

</p>

<p align="center">
  A lightweight Foundry VTT module that lets GMs and module developers instantly discover the data path of any field on an Item or Actor sheet.
</p>

<hr>

<h2>Why</h2>
<p>
  Ever needed the exact <code>system.xxx</code> path of a field — say, while translating an item or building a new module — and ended up exporting the whole Item just to dig through the JSON? This module removes that step entirely.
</p>

<h2>How it works</h2>
<ol>
  <li>Activate the inspector shortcut.</li>
  <li>Hover over any field on an open Item or Actor sheet.</li>
  <li>A tooltip shows the field's data path in real time.</li>
  <li>Click the field to copy the path to your clipboard.</li>
</ol>

<h2>Default Shortcut</h2>
<p>
  <kbd>Alt</kbd> + <kbd>P</kbd>
</p>
<p>
  Fully rebindable under Foundry's <strong>Configure Controls</strong> menu.
</p>

<h2>Who it's for</h2>
<ul>
  <li>🌐 <strong>Translators</strong> — quickly locate which field a piece of text belongs to.</li>
  <li>🛠️ <strong>Module developers</strong> — inspect any system's data structure without digging through compendium exports.</li>
  <li>🎲 <strong>GMs</strong> — build macros and automations faster.</li>
</ul>
