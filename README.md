<h1>Property Path Inspector | Inspetor de Caminho de Propriedades</h1>


<img src="https://img.shields.io/badge/Foundry-v13-ff6400?style=flat&labelColor=242526" alt="foundry-version">
<img src="https://img.shields.io/badge/Version-1.00-00247d?style=flat&labelColor=242526" alt="module-version">

</div>

<br>

<p>A lightweight Foundry VTT module that lets GMs and module developers instantly discover the data path of any field on an Item or Actor sheet.</p>
<p>This project is developed independently, in the developer's free time. Suggestions, bug reports, and fixes are always welcome.</p>

<br>

<p>Um módulo leve para o Foundry VTT que permite a GMs e desenvolvedores de módulos descobrir instantaneamente o caminho de dados de qualquer campo em uma ficha de Item ou Actor.</p>
<p>Este projeto é desenvolvido de forma independente e durante o tempo livre. Sugestões, correções e relatos de erros são sempre bem-vindos.</p>
